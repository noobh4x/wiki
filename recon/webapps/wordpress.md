# WordPress Reconnaissance

## Default Files

```
wp-login.php
wp-config.php
```

## Default Folders

```
/wp-content
/wp-content/plugins
/wp-content/themes
/wp-admin
```

## Determine Version

> Search the source code for generator, and look for a line similar to this

```html
<meta name="generator" content="WordPress 4.9.7" />
```

## Tools

* [WPScan](https://wpscan.org/)
